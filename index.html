Core Capabilities

Everything you need to regain control over your desktop environment.

Workspaces, not windows

MultiDesk focuses on organization, not window management. Each workspace has its own set of Windows shortcuts (.lnk) and an optional wallpaper.

  • Independent shortcuts
  • Optional wallpapers
  • Fast floating bar switch

High performance by design

Optimized for responsiveness. Workspaces load quickly, wallpaper files are handled safely (no file locks), and layouts are stored locally.

  • Lightweight runtime
  • Zero wallpaper file locks
  • Local-first storage

Designed for focused workflows

Customize your setup how you actually work. Your icon layout is saved per workspace, so organization stays consistent across restarts.

  • Saved icon layouts
  • Drag & drop + snap-to-grid
  • Easy shortcut management

Floating Workspace Bar

The dockable floating bar lets you switch workspaces in one click. Pin it to any screen edge, keep it always-on-top, and access Settings instantly. A tray icon provides quick actions and a clean “always running” experience.

Desktop-like Workspace Window

Each workspace is shown in a stable dedicated window that can run in full screen while keeping the Windows taskbar visible. This gives you a clean desktop-style surface without breaking standard Windows behavior.

Layout Persistence

Arrange icons with drag & drop. Snap-to-grid keeps the desktop clean and aligned. Layout is saved per workspace so it stays the same after switching or restarting.

Shortcut Management

MultiDesk works with standard Windows shortcuts (.lnk). Add shortcuts from Settings or by opening the workspace items folder. Right‑click shortcuts to copy or move them between workspaces, or delete links to the Recycle Bin.

FAQ

Does MultiDesk replace Windows Virtual Desktops?

No. MultiDesk is a workspace organizer for shortcuts and wallpapers. It does not move your open windows between desktops.

What shortcut types are supported?

MultiDesk displays standard Windows .lnk shortcuts.

Can I move/copy shortcuts between workspaces?

Yes. Use the right‑click menu on an icon to move or copy it to another workspace.

Where is my layout saved?

Each workspace stores its icon positions locally (layout persistence per workspace).
